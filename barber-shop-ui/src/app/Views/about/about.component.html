Sobre o projeto

Este projeto foi desenvolvido como parte do bootcamp Decola Tech 2025, promovido pela Avanade em parceria com a DIO. O projeto tem como objetivo criar um sistema fullstack de gerenciamento para uma barbearia utilizando Spring Boot e Angular. O sistema permite o agendamento de horários, gerenciamento de serviços e horários de funcionamento.
